# 画像素材について

このディレクトリには以下の画像を配置してください：

## 推奨画像

1. **neck-anatomy.svg** または **neck-anatomy.png**
   - 首の解剖図（頸椎、筋肉）
   - サイズ: 800x600px
   - フォーマット: SVG推奨、PNG可

2. **neck-stretch-right.svg** または **neck-stretch-right.png**
   - 右側屈のストレッチイラスト
   - サイズ: 400x400px
   - フォーマット: SVG推奨、PNG可

3. **neck-stretch-left.svg** または **neck-stretch-left.png**
   - 左側屈のストレッチイラスト
   - サイズ: 400x400px
   - フォーマット: SVG推奨、PNG可

4. **proper-posture.svg** または **proper-posture.png**
   - 正しい姿勢のイラスト
   - サイズ: 400x400px
   - フォーマット: SVG推奨、PNG可

## 画像がない場合

画像がない場合は、代わりに**絵文字**や**Font Awesome アイコン**を使用します。

## 無料素材サイト

- https://www.flaticon.com/ (アイコン、イラスト)
- https://undraw.co/ (SVGイラスト)
- https://www.freepik.com/ (医療イラスト)
